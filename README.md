# Student Management System

## 📌 Description
This project is a simple Student Management System built using Python. It allows users to add student details, calculate average marks, and identify the topper.

## 🚀 Features
- Add student details
- Calculate average marks
- Display all students
- Show topper

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Install Python
2. Run the file using:
   python student_management.py

## 📊 Output
- Student records displayed
- Topper identified based on average marks
